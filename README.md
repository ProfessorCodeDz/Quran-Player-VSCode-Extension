# Quran Player

#### Quran Player enables you to listen to the Quran with various Reciters and includes all Surahs.

1. **Crtl** + **Shift** + **P**
2. Choose **Quran Player**
3. Choose a **Reciter**
4. Choose a **Surah**
5. **Play**
